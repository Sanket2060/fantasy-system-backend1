What are sets?
A Set is not an Array, but it is a collection of unique values
const mySet = new Set([1, 2, 3, 3, 4]);
console.log(mySet); // Output: Set { 1, 2, 3, 4 }

const myArray = [1, 2, 3, 3, 4];
console.log(myArray); // Output: [1, 2, 3, 3, 4]
